Descrição

Este projeto teve como intuito realizar uma análise de dados à respeito do perfil do eleitor paraibano nas últimas eleições e como o mesmo mudou ao longos dos anos fazendo comparações desde das eleiçoes de 2000 até a última eleição. As variáveis consideradas para tal análise foram: SEXO, FAIXA-ETÁRIA e NÍVEL DE ESCOLARIDADE.
